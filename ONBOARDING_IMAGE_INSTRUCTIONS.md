# Running Legs Background Image

## Image Requirements

To complete the onboarding screen update, you need to add the running legs image to the Assets catalog.

### Location
`StepComp/Assets.xcassets/running-legs.imageset/`

### Required Files
1. `running-legs.jpg` (or .png) - 1x resolution (base size)
2. `running-legs@2x.jpg` (or .png) - 2x resolution (recommended)
3. `running-legs@3x.jpg` (or .png) - 3x resolution (recommended)

### Image Specifications
- **Subject**: Close-up of runner's legs in motion on a track/road
- **Orientation**: Portrait/vertical
- **Recommended Size**: 
  - 1x: ~414 x 896 px (iPhone base)
  - 2x: ~828 x 1792 px
  - 3x: ~1242 x 2688 px
- **Format**: JPEG or PNG
- **Quality**: High quality, sharp focus on legs
- **Colors**: Natural, athletic shoes visible (preferably blue/teal as shown in reference)

### How to Add Images

1. **Find or Download Image**
   - Use a stock photo from Unsplash, Pexels, or similar
   - Search terms: "runner legs track", "running shoes motion", "athlete legs running"
   - Example: The reference image shows blue running shoes on a track

2. **Add to Xcode**
   - Open `StepComp.xcodeproj` in Xcode
   - Navigate to `Assets.xcassets` in the file navigator
   - You'll see the `running-legs` image set already created
   - Drag and drop your images into the appropriate slots (1x, 2x, 3x)

3. **Alternative: Manual File Addition**
   - Place your image files directly in:
     `StepComp/Assets.xcassets/running-legs.imageset/`
   - Name them:
     - `running-legs.jpg`
     - `running-legs@2x.jpg`
     - `running-legs@3x.jpg`

### Recommended Free Image Sources
- [Unsplash](https://unsplash.com/s/photos/running-legs)
- [Pexels](https://www.pexels.com/search/running%20legs/)
- [Pixabay](https://pixabay.com/images/search/running/)
